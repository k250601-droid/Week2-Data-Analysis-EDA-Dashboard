# 📊 Week2-Data-Analysis-EDA-Dashboard

## Overview

This project was completed as part of the Week 2 Data Science Internship task. The objective was to perform Data Cleaning, Data Preprocessing, Exploratory Data Analysis (EDA), Data Visualization, and build an Interactive Dashboard using Streamlit.

The dataset used in this project is the **Netflix Movies and TV Shows Dataset**.

---

## Objectives

- Load and analyze a real-world dataset.
- Clean and preprocess the data.
- Perform Exploratory Data Analysis (EDA).
- Create meaningful visualizations.
- Build an interactive dashboard using Streamlit.
- Prepare the dataset for future Machine Learning tasks.

---

## Key Insights

- Movies are more common than TV Shows.
- Most Netflix content was released after 2015.
- TV-MA is the most frequent content rating.
- The United States contributes the highest number of titles.
- Drama is one of the most common genres on Netflix.
